## Installation

1. First, clone this repository

```bash
git clone https://github.com/KeysHD/portfolio.git
```

2. Install all the dependencies

```bash
yarn install
```

3. Run the project

```bash
yarn dev
```

4. Navigate to `https://localhost:3000`
